[Mysql]DML增删改操作

发表于2024-07-29|更新于2024-07-30|Mysql
|阅读量:

增加

  1. 单条数据：

    insert into 表名(字段1，字段2，字段3....) values(值1，值2，值3....)

    注意：传字符串时要手动加上单引号

  2. 多条数据：

    insert into 表名(字段1，字段2，字段3....) values

    (值1，值2，值3…),

    (值1，值2，值3…);

修改

  • update 表名 set 字段名1 = 值1，字段名2 = 值2，..... where(限定条件)；

删除

  • delete from 表名 条件限制
